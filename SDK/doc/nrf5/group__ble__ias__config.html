#define BLE_IAS_CONFIG_DEBUG_COLOR

ANSI escape code prefix.

  • 2 - Red
  • 3 - Green
  • 4 - Yellow
  • 5 - Blue
  • 6 - Magenta
  • 7 - Cyan
  • 8 - White
Note
This is an NRF_CONFIG macro.
