#define TASK_MANAGER_CONFIG_STACK_GUARD

Configures stack guard.

Following options are available:

  • 0 - Disabled
  • 4 - 32 bytes
  • 5 - 64 bytes
  • 6 - 128 bytes
  • 7 - 256 bytes
  • 8 - 512 bytes
